# GameNetwork - Социальная сеть для игроков

## Описание
Социальная сеть для геймеров с системой лайков, комментариев и друзей.

## Функционал
- Пользовательская система
- Профили
- Лента постов
- Система лайков
- Комментарии
- Система друзей
- Поиск и фильтрация
- Защита и безопасность


## Технологии
- PHP 8.0+ – основной язык серверной части
- MySQL 8.0 – система управления базами данных
- PDO (PHP Data Objects) – безопасная работа с БД
- REST-like API – JSON API для взаимодействия с клиентом
- HTML5, CSS3 – семантическая верстка и стили
- JavaScript (ES6+) – интерактивность и AJAX-запросы
- Docker – контейнеризация приложения

## Установка
1. Установите Docker и Docker Compose
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Ykianik089/game-social-network.git
   cd game-social-network
3. Запустите контейнеры:
    docker-compose up -d
4. Откройте в браузере: http://localhost:8080
